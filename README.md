# XmlIterator

Fast and memory efficient XML iterator based on XmlReader.

[![Build Status](https://travis-ci.org/Koc/xml-iterator.svg?branch=master)](https://travis-ci.org/Koc/xml-iterator)

## Installation

Run

```
$ composer require brouzie/xml-iterator
```
